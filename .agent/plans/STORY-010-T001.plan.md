## Summary

Add IP-based rate limiting to all auth endpoints: login (10 req/15 min), register (5 req/1 hour), refresh (20 req/15 min). Redis-backed counters with INCR/EXPIRE. Graceful degradation: if Redis is unavailable, allow requests through. All 429 responses include Retry-After header.

**Agent instruction:** Write the test demonstrating missing rate limiting FIRST. Confirm it fails. Then implement.
