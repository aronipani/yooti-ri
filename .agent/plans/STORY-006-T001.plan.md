## Summary

Build the checkout backend: Order and OrderItem models, POST /api/v1/orders endpoint with server-side price recalculation, atomic stock decrement (SELECT FOR UPDATE), payment gateway integration (interface), idempotency key to prevent duplicate orders, confirmation email dispatch. Handle concurrent purchase race conditions.
