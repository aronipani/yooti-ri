## Summary

Build the authentication API: POST /auth/login (validate credentials, issue JWT + refresh token in httpOnly cookie), POST /auth/logout (server-side session invalidation), POST /auth/refresh (rotate token). Implement account lockout after 10 failed attempts (30 min). JWT expiry 24h.
