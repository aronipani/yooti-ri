## Implementation Steps
1. Write failing unit tests for Product model (fields, constraints, defaults)
2. Write failing unit tests for Category model (fields, slug uniqueness)
3. Create `category.py` model — id, name, slug (unique), timestamps from Base
4. Create `product.py` model — id, name, description, price (Numeric 10,2), stock_quantity (int, default 0), category_id (FK), images (JSON array of URLs), is_active (bool, default true), timestamps from Base
5. Update `models/__init__.py` to export both models
6. Create Alembic migration
7. Create seed script with >=25 products across >=3 categories
8. Run lint → types → tests → verify all green
